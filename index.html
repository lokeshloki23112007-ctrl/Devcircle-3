Our goals

Find the first 50 members

02

Start small study groups

Once there are enough members, small peer groups around shared interests — web dev, DSA, AI — instead of one big generic chat.

Ship a first community project

Not a hackathon, not a big launch — just one small project built together, so the "0 projects" above stops being zero.
